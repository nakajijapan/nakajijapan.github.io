wordpress_logpiプチ改造

January 09, 2010 ( wordpress )

画像表示したかったのでプチ改造

ループ中に以下の処理をいれて、画像を小さく表示できるようにした。

// 置換                                                                                                                                                                  
 $message['description'] = preg_replace("/img\.logpi\.jp\/(.+)_l\.(jpg|png|gif)\">/", "img.logpi.jp/$1_m.$2\">
", $message['description']);

でも画像用のAPIとかほしいよね〜。
