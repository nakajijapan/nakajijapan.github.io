June 05, 2011 ( Web )

欲求不満Chromeアプリ

なんか最近はやってみるみたいだからチョッパやで作ってだしてみました。

うん、べつにたいしたことはしていないですよ。 なので、とりあえずはギガンティックαバージョンです。（ごめんなさい）

もっとブラウザのネイティブな処理をつかっておもしろいことできたらと 思っているので勉強がてらもっとアプリ的なものを作ります。

作るコンセプトは「リアルタイム」と「暇つぶし」かな。 いろいろ渦巻いている情報を無駄に見ることができたり、無駄に視覚的にしたり、みてるやつほんと暇なんじゃねーかって言う感じのものです。

あ、hostedじゃなくていいかもしれない。

ということで今後も地道にやっていこうっと！
