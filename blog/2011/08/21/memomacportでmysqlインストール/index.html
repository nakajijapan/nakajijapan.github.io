[memo]macportでmysqlインストール

August 21, 2011 ( diary )

インストール

sudo port install mysql5-server

Mysql初期化

sudo -u mysql mysql_install_db5

Mysql起動

sudo /opt/local/share/mysql5/mysql/mysql.server start

Mysql自動起動設定

# -wは設定ファイルが無効になっていても,Disableが有っても読み、ファイルのDisableを削除します。
sudo launchctl load -w /Library/LaunchDaemons/org.macports.mysql5.plist

自動起動設定は「init.d」的な機能は利用していないので迷ってしまいましたよ。BSDベースはちょっとわからんです。

my.cnfの移動

sudo cp /opt/local/share/mysql5/mysql/my-small.cnf /opt/local/etc/mysql5/my.cnf

■参照URL http://macwiki.sourceforge.jp/wiki/index.php/%E3%82%B3%E3%83%9E%E3%83%B3%E3%83%89/launchctl
