Vagrant勉強会でLTした

February 28, 2014 ( vagrant )

社内でVagrant勉強会があったので参加・発表してきました。

僕の他にも公開しているメンバーがいるのでご紹介します。 こちらのブログにもリンクがあるので他の人の資料にもたどりつけると思います。
