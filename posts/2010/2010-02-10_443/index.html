DSOでmod_perlをインスコ

February 10, 2010 ( Apache )

mod_perをインスコログ

これダウンロード先 http://perl.apache.org/download/index.html

# 解答
$ tar zxvf mod_perl-2.0-current.tar.gz
$ cd mod_perl-2.0.4/
 
# モジュールの確認を行う
$ httpd -l
Compiled in modules:
  core.c
  prefork.c
  http_core.c
  mod_so.c

$ perl Makefile.PL USE_APXS=1 WITH_APXS=/usr/local/apache/bin/apxs EVERYTHING=1 MP_APR_CONFIG=/usr/local/apache/bin/apr-1-config
$ make
$ make install

$ vi httpd.conf LoadModule perl_module modules/mod_perl.so
