[復習]javascriptでオブジェクト指向　その２

May 01, 2010 ( Diary )

さっきの続き。 継承できるようにする。

// 小クラス
function TestChild() {};
TestChild.prototype = new Test();
TestChild.prototype.setStr = function (str) {
    this._str = str + "(child)";
}

まずは、「prototype」で親クラスをインスタンス化する。 オーバライドするメソッドだけ宣言して処理内容を記述する！
